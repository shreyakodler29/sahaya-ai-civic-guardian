# Sahaya AI Civic Guardian

## Overview
AI system to monitor and protect public infrastructure using computer vision and machine learning.

## Features
- Damage Detection (YOLOv8)
- Maintenance Prediction
- Civic Integrity Score

## OpenEnv Compliance
- step()
- reset()
- state()
- reward function

## How to Run
pip install -r requirements.txt
python inference.py

## Deployment
- Hugging Face (app.py)
- Docker supported

## Impact
Helps improve rural governance and public asset protection.
